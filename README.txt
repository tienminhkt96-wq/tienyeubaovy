Hướng dẫn nhanh - Phiên bản final 'Gửi đến Bảo Vy yêu dấu' (nền hoa đỏ đậm, tim bay, nhạc Perfect tự chạy)

Các file gồm:
- index.html
- style.css
- script.js

Cách dùng nhanh:
1. Giải nén zip.
2. Upload 3 file (index.html, style.css, script.js) vào repository GitHub của anh, ở thư mục gốc.
   - Hoặc dùng Upload Folder trên Vercel khi tạo project mới.
3. Vercel sẽ tự build và trả về link dạng https://<project-name>.vercel.app
   - Để có tên https://tienyeuvy.vercel.app, đặt tên project là 'tienyeuvy' khi deploy (nếu tên chưa ai lấy).
4. Lưu ý: Một số trình duyệt chặn autoplay nhạc. Nếu nhạc không tự phát, hãy click 1 lần trên trang để cho phép playback.

Nếu anh muốn, em có thể trực tiếp **hỗ trợ kiểm tra repo** nếu anh gửi link GitHub — em sẽ nói chính xác chỗ bấm để redeploy.
